# 1월 14일 파이썬

#### sys모듈

- 파이썬 인터프리터(엔진)가 실행되는 환경과 해석기의 여러 기능 조회 및 관리
- `sys.version` : 버전 정보
- `sys.path` : path 정보
- `sys.exit(0)` : 프로그램 종료
  - 인수가 0이면 정상 종료, -1이면 비정상 종료



#### 명령행 인수(command line argument)

- 프로그램 안에어 input으로 받지 않아도 실행하면서 명령행 인수를 줄 수 있음
- 시작하면서 1번만 받을 수 있음

- `copy a.txt b.txt`

- `sys.argv` : 명령행 인수를 읽어옴
